# Contributing to Awesome AI Agents

## Guidelines

- Only add tools, frameworks, or papers that are **actively maintained** (updated within last 6 months)
- Each entry needs: **Name + Link + One-line description**
- Sort entries **alphabetically** within each section
- No duplicates across sections
- Research papers must be **peer-reviewed or widely cited** (>50 citations or from top venues)
- No promotional content or paid placements

## How to Contribute

1. Fork this repository
2. Add your entry in the appropriate section
3. Submit a PR with a brief explanation of why it belongs here

## Quality Checks

Before submitting, verify:
- [ ] Link works and is not behind a paywall (for tools/frameworks)
- [ ] Project has >100 GitHub stars OR is from a major organization
- [ ] Description is concise (max 15 words)
- [ ] Correct alphabetical placement
